# Программа для расчета эпюр балок

Консольное приложение на Python для строительной механики, которое рассчитывает и интеративно отображает эпюры внутренних усилий (Nx, Qy, Mz) в стержневых системах.

## Требования
- Python 3.11.9 (или выше)

## Инструкция по установке

Чтобы запустить приложение на своем компьютере, выполните следующие шаги:

1. **Склонируйте репозиторий** (или скачайте исходный код проекта и перейдите в его папку) в консоли:
   ```bash
   git clone https://github.com/ZAGAIQ/balkiprog
   cd BalkiProga
   ```

2. **Создайте виртуальное окружение**, чтобы изолировать зависимости проекта:
   ```bash
   py -m venv venv
   ```

3. **Активируйте виртуальное окружение**:
   - На **Windows** (в командной строке или PowerShell):
     ```bash
     venv\\Scripts\\activate
     ```
   - На **macOS / Linux**:
     ```bash
     source venv/bin/activate
     ```

4. **Установите все необходимые зависимости**:
   ```bash
   pip install -r requirements.txt
   ```

5. **Запустите программу**:
   ```bash
   py calc_epures.py
   ```

## Использование
Данные можно вводить двумя способами после запуска программы:
1. **Вручную через консоль**
2. **Через файл** — отредактируйте файл `task.txt` (с параметрами вашей балки) и выберите пункт `2` при запуске приложения.

Если вы все сделали правильно, программа посчитает уравнения и построит графики в отдельном окне.
